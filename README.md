
# dothewordpress

Es un conjunto de script que levantar un sitio en wordpress usando docker y docker-compose. En si el script funciona haciendo un git clone. 

```git clone https://github.com/PabloCarrai/dothewordpress.git```

Esto te genera una carpeta con el nombre del proyecto, ingresas en la misma. 

```cd dothewordpress```

y ejecutas la crecion del proyecto. 
```./dothewordpressturururu.sh nombresitio.org.ar```

Esto te creara una carpeta con el nombresitio.org.ar ingresas en el

```cd nombresitio.org.ar```
y levantas el wordpress
```docker-compose up -d```
